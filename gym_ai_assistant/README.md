# AI Gym & Nutrition Assistant

## Project Description
This project is an AI-assisted fitness and nutrition recommendation system developed in Python.

The system receives user information such as:
- age
- weight
- height
- fitness goal

and generates:
- BMI calculation
- estimated daily calorie recommendation
- workout recommendation

## Features
- BMI Calculator
- Daily Calorie Estimator
- Workout Recommendation System
- User Input Handling
- AI-assisted decision workflow

## Technologies Used
- Python
- Visual Studio 2022
- pytest

## How to Run
1. Open the project in Visual Studio 2022
2. Run the Python file
3. Enter the requested user information
4. Receive personalized recommendations

## Example Goals
- fat loss
- muscle gain
- maintenance

## Testing
The system can be tested using different user scenarios and input values.

## Deployment Strategy
This project is designed as a local desktop Python application.
