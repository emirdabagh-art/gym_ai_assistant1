# AI Gym & Nutrition Assistant

## Project Description

AI Gym & Nutrition Assistant is a Python-based intelligent fitness recommendation system.

The system receives user information such as:
- age
- weight
- height
- fitness goal

Using this information, the application performs:
- BMI calculation,
- daily calorie estimation,
- workout recommendation generation.

The purpose of the project is to demonstrate an AI-assisted software solution that integrates tool-based calculations and decision-making logic.
